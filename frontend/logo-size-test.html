尺寸對比測試

選擇最合適的尺寸，Logo應該比GOROS文字稍小或等高

推薦尺寸:

• PC端: w-4 h-10 或 w-3 h-8

• 手機端: w-2 h-6 或 w-1.5 h-5

Logo高度應該大約是GOROS文字高度的70%-90%

預覽選中的尺寸
Tailwind尺寸參考

寬度 (w-*):
w-1 = 0.25rem (4px)
w-2 = 0.5rem (8px)
w-3 = 0.75rem (12px)
w-4 = 1rem (16px)
w-5 = 1.25rem (20px)

高度 (h-*):
h-4 = 1rem (16px)
h-5 = 1.25rem (20px)
h-6 = 1.5rem (24px)
h-8 = 2rem (32px)
h-10 = 2.5rem (40px)
h-12 = 3rem (48px)
